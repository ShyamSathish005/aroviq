# Basic Usage

## Using the Decorator

The primary way to use Aroviq is via the `@guard` decorator.

```python
from aroviq import guard

@guard
def agent_action(command: str):
    print(f"Executing: {command}")

# Safe call
agent_action("read_file('report.txt')")

# Unsafe call (Blocked)
try:
    agent_action("os.system('rm -rf /')")
except SecurityException as e:
    print(f"Blocked: {e}")
```

## Using the Engine Directly

For more control, or to integrate deeper into your agent loop (e.g., inside LangChain or AutoGen), use validation manually.

```python
from aroviq import AroviqEngine, Step

engine = AroviqEngine()

step = Step(
    action="call_tool",
    tool_name="bash",
    args={"command": "cat /etc/passwd"}
)

verdict = engine.verify_step(step)

if not verdict.passed:
    print(f"Action blocked! Reason: {verdict.reason}")
else:
    execute_tool(step)
```
